## 简介：
本项目适用于基于**Clash Meta Core**的[**OpenClash**](https://github.com/vernesong/OpenClash)插件配置文件示例，涵盖部分规则集及GeoSite、GeoIP数据库，自动更新。

## 说明：

> 使用本项目配置文件前，参照此[**视频**](https://youtu.be/O5FQ3MOcoqU)进行相应设置
- 一切配置皆yaml
- 多机场与自建节点融合使用
- 自动定期最新规则组
