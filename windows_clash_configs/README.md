## 简介：

## 一键启动脚本教程：
1. 保存这些文件到本地电脑D盘的某个文件夹中，文件夹可自行命名，config.yaml文件参考教程文档修改
2. 鼠标右键clash.meta-windows-amd64.exe 然后打开属性 -> 兼容性 -> 勾选以管理员身份运行此程序
3. 鼠标右键clash.bat 发动到 -> 桌面快捷方式
4. 下次就可以双击桌面上的clash.bat就可以一键启动了
